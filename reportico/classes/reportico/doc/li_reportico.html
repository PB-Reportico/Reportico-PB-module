The Reportico User Manual provides details of how to install and configure Reportico, how to create and run Reportico reports, how to create report menus.

Tutorial

The Reportico Tutorial tells you how to run through the supplied example tutorials. The tutorials explain how to create the DVD Library database. The supplied tutorials will take you through creation of a new report, adding criteria, groups, graphs, assignments, custom code and more.
